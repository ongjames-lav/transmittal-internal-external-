# "Picked" View Added

I have added the **"PICKED"** filter button to:
1.  **User Sent Items** (Inbox - Sent Folder)
2.  **Custodian Outgoing Items**

**Verification:**
- **Users**: Go to `/transmittals/sent/` and click the "PICKED" tab.
- **Custodians**: Go to `/transmittals/custodian/outgoing/` and click the "PICKED" tab.

You should now see all transmittals that have been picked up by the receiver but not yet marked as fully received.
