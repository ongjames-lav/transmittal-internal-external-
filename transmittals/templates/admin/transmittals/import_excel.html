⚠️ IMPORTANT: Department ≠ Location

Departments are user job departments (ACCOUNTING, EXECUTIVE, HRAD, DISTRIBUTION, etc.)
Locations are physical warehouse/office locations (see list below)
These are completely different fields!

Origin Location and Destination Location must use the 5 assigned physical locations ONLY.

📋 How to Use

  1. Prepare your Excel file with the columns listed below
  2. Make sure all Required fields are filled
  3. Use Location names/codes for Origin and Destination (see list below, NOT department names)
  4. Upload the file using the form below
  5. Review any errors or warnings
  6. Successfully imported records will be added to the database
